# Contribute

**``socca``** is an open-source library (under active development) that has grown thanks to feedback and contributions from many students and fellow researchers. As such, any contributions from the community are greatly appreciated, be it in the form of bug reports, feature requests, or direct code edits. For detailed instructions on how to contribute, please refer to the official [contributing guidelines](https://github.com/lucadimascolo/socca/blob/main/CONTRIBUTING.md).

If you are unsure where to start, opening an issue on the [GitHub issue tracker](https://github.com/lucadimascolo/socca/issues/new) or a [discussion thread](https://github.com/lucadimascolo/socca/discussions/new) to share ideas or ask questions is always encouraged.
